## Update instructions

https://sourcethemes.com/academic/docs/update/#if-you-installed-academic-kickstart

```

cd themes/academic
git checkout <VERSION>

```

## Add post

[Create a blog post](https://sourcethemes.com/academic/docs/managing-content/#create-a-blog-post)

```

```
