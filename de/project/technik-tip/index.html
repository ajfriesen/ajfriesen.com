Code Webseite

Eine simple kleine Webseite für mein kleines Nebenprojekt.

Das Ziel ist es tägliche IT Probleme für jederman einfach zu erklären. Ein ähnliches Projekt im Bereich der Finanzen gibt es schon mit www.finanztip.de welches dieses hier bestimmt auch ein wenig geprägt hat.

Techstack

  • Die Webseite wird mit dem statischem Webseiten Generator Hugo gebaut
  • Ich nutze das Hugo academic theme
  • Für das Hosting nutze ich GitHub Pages
  • Mit Github Actions habe ich das Deployment der Webseite komplett automatisiert

Schaut gerne mal vorbei: https://www.technik-tip.de/
